inspect¶

auxjad.inspect(client)¶

Makes an inspection agent. See Inspection for the documentation of all of its methods.

Example:

>>> staff = abjad.Staff("c'4 e'4 d'4 f'4")
>>> auxjad.inspect(staff[2:])
Inspection(client=Selection([Note("d'4"), Note("f'4")]))

Note

Auxjad automatically adds all methods of Inspection as extension methods to abjad.inspect(). Therefore they can be used from either auxjad.inspect() or abjad.inspect(), as shown below:

>>> container = abjad.Container(r"c'4 d'4 e'4 f'4")
>>> auxjad.inspect(container[:]).selection_is_full()
True
>>> abjad.inspect(container[:]).selection_is_full()
True
