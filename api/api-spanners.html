spanners¶

The spanners subpackage contains functions that add spanners to containers and selections.

Note

These functions are imported directly into the auxjad namespace. Therefore, to use a function such as auxjad.half_piano_pedal() you only need to import auxjad and use it directly from auxjad’s namespace:

>>> import abjad
>>> import auxjad
>>> staff = abjad.Staff(r"c'''4 d'''4 e'''4 f'''4")
>>> auxjad.half_piano_pedal(staff[:], 1)

These functions are also added to the abjad namespace:

>>> import abjad
>>> import auxjad
>>> staff = abjad.Staff(r"c'''4 d'''4 e'''4 f'''4")
>>> abjad.half_piano_pedal(staff[:], 1)
