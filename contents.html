The auxjad’s API¶

Below is a table with all classes and functions included in auxjad. Click on their their names or use the left side panel to navigate to the individual documentaion of each class and function.

auxjad.are_containers_equal(container1, …) A comparator function returning True when two containers are identical and False when they are not.
auxjad.are_leaves_tieable(leaf1, leaf2) A comparator function returning True when two leaves have identical pitches and thus can be tied, otherwise returning False.
auxjad.ArtificialHarmonic(*arguments, …) Creates an abjad.Chord with a tweaked top note head for notating artificial harmonics.
auxjad.CartographyContainer(container, *, …) A container used to store, manipulate, and select objects using a decaying weighted function.
auxjad.close_container(container) Changes the time signature of the last bar of an underfull abjad.Container in order to make it full, if necessary.
auxjad.fill_with_rests(container) Fills an abjad.Container with rests in order to make it full.
auxjad.HarmonicNote(*arguments, multiplier, …) Creates an abjad.Note with tweaked notehead for harmonics.
auxjad.is_container_full(container) Checks if an abjad.Container is full.
auxjad.LeafDynMaker(*, increase_monotonic, …) An extension of abjad.LeafMaker which can also take optional lists of dynamics and articulations.
auxjad.LeafShuffler(container, *, …) Takes an input abjad.Container and shuffles its leaves.
auxjad.LoopWindow(container, *, window_size, …) Using a looping window, this slices an input abjad.Container and output them as containers.
auxjad.LoopWindowByElements(container, *, …) Takes an abjad.Container as input as well as an integer representing the number of elements per looping window, then outputs a container with the elements processed in the looping process.
auxjad.LoopWindowByList(container, *, …) Similar to LoopWindowByList, but instead of taking an abjad.Container input, it takes a list of arbitrary size.
auxjad.remove_repeated_dynamics(container, …) A function which removes all consecutive repeated dynamics.
auxjad.remove_repeated_time_signatures(container) A function which removes all unecessary time signatures.
auxjad.respell_container(container, *, …) Applies auxjad.respell_chord to all abjad.Chord of an abjad.Container.
auxjad.respell_chord(chord, *, …) Changes the accidentals of individual pitches of an abjad.Chord in order to avoid augmented unisons.
auxjad.rests_to_multimeasure_rest(container) Takes an abjad.Container and converts all rests into abjad.MultimeasureRest’s when possible.
auxjad.simplified_time_signature_ratio(…) A function simplifies the ratio of a given time signature respecting a minimum denominator value.
auxjad.sync_containers(*containers, …) Takes an arbitrary number of abjad.Container’s (or child classes), finds the longest one and adds rests to all the shorter ones, making them the same length.
auxjad.TenneysContainer(container, *, …) TenneysContainer in an implementation of the Dissonant Counterpoint Algorithm by James Tenney.
auxjad.underfull_duration(container) Checks if an abjad.Container is underfull and returns the missing abjad.Duration.
