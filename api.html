auxjad API¶

Below is a table with all classes and functions included in auxjad. Click on their their names or use the left side panel to navigate to the individual documentaion of each class and function.

auxjad.adjust_clefs(container, *[, …])

Mutates an input container (of type abjad.Container or child class) in place and has no return value.

auxjad.ArtificialHarmonic(*arguments, …)

Creates a chord with a tweaked top note head for notating artificial harmonics.

auxjad.CartographySelector(contents, *, …)

A selector used to store, manipulate, and select objects using a weighted function constructed with a fixed decay rate.

auxjad.close_container(container)

Mutates an input container (of type abjad.Container or child class) in place and has no return value.

auxjad.container_is_full(container)

Returns a bool representing whether an input container (of type abjad.Container or child class) has its last bar is fully filled in or not.

auxjad.containers_are_equal(container1, …)

Returns a bool representing whether two input containers (of type abjad.Container or child class) are identical or not.

auxjad.enforce_time_signature(container, …)

Mutates an input container (of type abjad.Container or child class) in place and has no return value.

auxjad.Fader(contents, *, fader_type, …)

This class can be used to fade in or fade out an abjad.Container by gradually removing or adding its logical ties one by one.

auxjad.fill_with_rests(container)

Mutates an input container (of type abjad.Container or child class) in place and has no return value.

auxjad.HarmonicNote(*arguments, multiplier, …)

Creates a note with tweaked notehead for harmonics.

auxjad.Hocketer(contents, *, n_voices, …)

Hocketer is a hocket generator.

auxjad.LeafDynMaker(*, increase_monotonic, …)

This class can be used to create leaves and logical ties from input lists of pitches, durations, dynamics, and articulations.

auxjad.leaves_are_tieable(leaf1, leaf2)

Returns a bool representing whether or not two input leaves (of type abjad.Leaf or child class) have identical pitch(es) and thus can be tied.

auxjad.LoopByList(contents, *, window_size, …)

This class can be used to output slices of a list using the metaphor of a looping window of a constant number of elements.

auxjad.LoopByNotes(contents, *, window_size, …)

This class can be used to output slices of an abjad.Container using the metaphor of a looping window of a constant number of elements.

auxjad.LoopByWindow(contents, *, …)

This class can be used to output slices of an abjad.Container using the metaphor of a looping window of a constant size (given by an abjad.Duration).

auxjad.Phaser(contents, *, step_size, float, …)

Phaser will shift all leaves of an abjad.Container by a fixed amount.

auxjad.remove_empty_tuplets(container)

Mutates an input container (of type abjad.Container or child class) in place and has no return value.

auxjad.remove_repeated_dynamics(container, *)

Mutates an input container (of type abjad.Container or child class) in place and has no return value.

auxjad.remove_repeated_time_signatures(container)

Mutates an input container (of type abjad.Container or child class) in place and has no return value.

auxjad.respell_container(container, *[, …])

Mutates an input container (of type abjad.Container or child class) in place and has no return value.

auxjad.respell_chord(chord, *[, …])

Mutates an input chord (of type abjad.Chord or child class) in place and has no return value.

auxjad.rests_to_multimeasure_rest(container)

Mutates an input container (of type abjad.Container or child class) in place and has no return value.

auxjad.Shuffler(contents, *, pitch_only, …)

Shuffler takes an input abjad.Container and shuffles or rotates its logical ties or pitches.

auxjad.simplified_time_signature_ratio(ratio, *)

Returns an abjad.TimeSignature with the simplified ratio of an input ratio according to a minimum denominator value.

auxjad.sync_containers(*containers[, …])

Mutates two or more input containers (of type abjad.Container or child class) in place and has no return value.

auxjad.TenneySelector(contents, *, weights, …)

This in an implementation of the Dissonant Counterpoint Algorithm by James Tenney.

auxjad.time_signature_extractor(container, *)

Returns a list with the abjad.TimeSignature for all measures of an input abjad.Container.

auxjad.underfull_duration(container)

Returns the missing abjad.Duration of an underfull container (of type abjad.Container or child class).
